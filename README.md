### Commit Type
- [ADD] : 파일, 기능 추가
- [REMOVE] : 파일, 기능 제거
- [UPDATE] : 파일, 기능 수정
- [REFACTOR] : 기능 변경 없이 동작 구조만 수정
- [STYLE] : 기능 변경 없이 UI만 수정
- [FIX] : 버그 수정
- [CHORE] : build, manifest 수정
- [TEST] : test 코드 작성
